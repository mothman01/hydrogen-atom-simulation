# Security Policy

## Supported Versions

Only the latest release of Atom Wavefunction Simulator receives security fixes.

| Version | Supported |
|---------|-----------|
| Latest  | ✅ |
| Older   | ❌ |

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report vulnerabilities privately by emailing **michaelcogan497@gmail.com** with:

1. A description of the vulnerability and its potential impact.
2. Steps to reproduce or a proof-of-concept (if applicable).
3. Any suggested mitigations you have in mind.

You can expect an acknowledgement within **72 hours** and a resolution timeline
within **14 days** for confirmed issues. We will credit reporters in the release
notes unless you request anonymity.
